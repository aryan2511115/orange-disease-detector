About This Project

Project Objective

This system uses advanced artificial intelligence and deep learning to detect diseases in orange fruits and leaves. It provides farmers with instant recommendations for treatment, prevention, and proper fertilizer usage.

Technology Stack

  • Backend: Flask, Python, TensorFlow
  • Frontend: HTML, CSS, JavaScript
  • Database: MySQL
  • ML Models: MobileNetV2, ResNet50, EfficientNetB0 (Transfer Learning)
  • Explainability: Grad-CAM

Detected Diseases

  • ✓ Citrus Canker (बैक्टीरिया रोग)
  • ✓ Black Spot (काला धब्बा)
  • ✓ Citrus Greening / HLB (साइट्रस ग्रीनिंग)
  • ✓ Leaf Miner (पत्ती माइनर)
  • ✓ Healthy Orange Leaf (स्वस्थ पत्ती)

Key Features

  • ✓ High Accuracy Detection - 95%+ accuracy using transfer learning
  • ✓ Disease Severity Analysis - Percentage-based severity assessment
  • ✓ Explainable AI - Grad-CAM visualizations
  • ✓ Multilingual Support - English, Hindi, Marathi
  • ✓ Prediction History - Track all predictions
  • ✓ Smart Recommendations - Treatment, prevention, fertilizer

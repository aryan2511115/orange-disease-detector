## 📞 Support & Debugging

**If You Encounter Issues:**

1. **Model not loading**: Check if `models/best_model.h5` exists
2. **Database error**: Verify MySQL is running and credentials are correct
3. **API not responding**: Check if Flask backend is running on port 5000
4. **Frontend not loading**: Verify templates/index.html path is correct
5. **CORS errors**: Check CORS configuration in app.py
